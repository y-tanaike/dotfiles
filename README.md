# インストール手順

#### 設定先に遷移
```
cd ~
git clone https://github.com/y-tanaike/dotfiles.git
cd dotfiles
```

#### 自分の設定をgithubからダウンロード
```
bash download.sh
```
####  設定を反映する(シンボリックリンクを割り当てる)
```
bash install.sh
```
